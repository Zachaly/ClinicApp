# ClinicApp
University project created for .NET programming course, created with .NET 10 ASP.NET Core backend, SQL Server Database and Vue3 on frontend

# Launching
Go into `ClinicApp` directory and run `docker compose up` command, requires Docker installed.

Frontend should be available on `http://localhost:3000` and backend on `http://localhost:5000`

# Tests
To run tests go into `ClinicApp` directory and run command `dotnet test` for all tests in solution or specify approporiate project file to run Unit/Integration tests separately
